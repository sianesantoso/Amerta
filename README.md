# Amerta - Web Kasir

Amerta is a web-based cashier system built using Flask. This project was developed during an internship to help manage store transactions, products, and cashier activities through a simple and efficient interface.

## Features

- Product management
- Cashier transaction system
- Database integration using SQL
- Simple and responsive UI
- Store logo and custom styling
- Secure password generation utility

## Tech Stack

- Python
- Flask
- HTML
- CSS
- SQL Database
